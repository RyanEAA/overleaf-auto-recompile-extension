# Auto Recompile Chrome Extension

Automatically clicks the "Recompile" button when the Overleaf editor changes.

## Features
- Watches for text changes
- Debounced auto-recompile
- Works on Overleaf

## Installation
1. Clone this repo
2. Go to chrome://extensions/
3. Enable Developer Mode
4. Load unpacked → select this folder

## Tech
- Chrome Extension (Manifest V3)
- MutationObserver
